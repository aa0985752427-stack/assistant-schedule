# 助理排班系統 🌸

診所助理的每月排班工具。純前端、免資料庫、免登入，開瀏覽器即可使用。

## 功能

- **每月月曆排班**：每天分**早班（9:30–18:30）**與**晚班（11:30–20:30）**。
- **一鍵指派**：每班顯示 5 位助理色塊（霞、嵐、偲、琳、阡），點一下即指派 / 取消。
- **醫師固定班參考**：日曆上標記醫師出勤日，方便安排助理人力。
  - 陳醫師：週一早班、週三 / 週五整天
  - 宇醫師：週一 14:00–20:30、週四整天
- **每週工時統計**：每班以 8 小時計（已扣午休 1 小時），自動對照**每人每週 40 小時**目標，並以顏色提示不足 / 剛好 / 超時。
- **套用醫師班表建議**：依醫師出勤日，自動在尚無人的班別帶入建議助理（可再手動調整）。
- **匯出文字 / Email 通知**：一鍵匯出班表文字檔，或透過預設信箱寄出。
- **自動儲存**：資料存於瀏覽器 `localStorage`，不會上傳。

## 本機使用

直接用瀏覽器開啟 `index.html` 即可，不需安裝任何東西。

（若要避免瀏覽器對本機檔案的限制，也可用任一靜態伺服器，例如 Python：）

```bash
python -m http.server 8080
# 開啟 http://localhost:8080
```

## 部署到 Zeabur

本專案是純靜態網站，Zeabur 會自動辨識。

1. 將程式碼推上 GitHub。
2. 到 [Zeabur](https://zeabur.com) 建立專案 → **Deploy from GitHub** → 選擇此 repo。
3. Zeabur 會依 `zbpack.json` 以「靜態網站」方式部署（根目錄即輸出目錄）。
4. 部署完成後綁定網域即可。

### 推上 GitHub（首次）

```bash
git init
git add .
git commit -m "初始版本：助理排班系統"
git branch -M main
git remote add origin https://github.com/<你的帳號>/<你的repo>.git
git push -u origin main
```

## 後續可加強

- **Email 通知升級**：目前使用 `mailto:`（開啟使用者信箱軟體）。若要「系統自動寄信」，可接 [EmailJS](https://www.emailjs.com/)（純前端、免後端）。
- 午班（若之後需要三班制）、每位助理的休假 / 偏好設定、匯出 PDF、多人雲端共用班表等。

## 檔案結構

```
index.html    單一自帶檔案（HTML + 淡粉紅樣式 + 排班邏輯，localStorage 儲存）
zbpack.json   Zeabur 靜態部署設定
```

> 為確保在各種環境（含預覽面板、直接雙擊開啟）都能正常顯示樣式，CSS 與 JS 已內嵌於 `index.html`。
